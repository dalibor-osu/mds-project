Jak spustit projekt:

1. Jestliže neexistují, vytvořte složku "temp" a "screenshots" v mateřském adresáři.
2. Přes PowerShell spusťte NGINX.exe.
3. Přes další PowerShell spusťte skript "stream_run_powershell.ps1".
4. V prohlížeči otevřete "localhost".

Správné načtení všech kamer může chvíli trvat. Pokud přehrávače ukazují chybu, zkuste chvíli počkat
a následně stránku znovu načíst.
